== Description ==

Stern Prescription Upload And Collect is a powerful plugin for pharmacies. Customer Upload prescription, pay online and collect it in pharmacy. 
It is a click & collect for pharmacies.
This plugin requires WooCommerce.


== Installation ==

e.g.

1. Upload zip to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Install pages and products
4. Place shortcodes in your templates
[Stern_Upload_And_Collect_upload_button type_document="prescription"]
[Stern_Upload_And_Collect_dashboard type_user="cust" statut_shortcode="50"]
[Stern_Upload_And_Collect_dashboard type_user="admin"] // deprecated
[showPrescriptionsListAdmin]
[sitepoint_contact_form]
[Stern_Upload_And_Collect_upload_button type_document="mutuelle"]
[showMutuelle_shortcode]
[showMutuelleAdminShortcode]
[showPrescriptionsListAdmin]
